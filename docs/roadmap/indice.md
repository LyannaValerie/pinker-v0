# Índice do roadmap

- **Classe:** Ponte
- **Papel:** navegação
- **Status:** ativo

Este índice roteia os blocos do roadmap sem substituir o topo executivo nem competir com os shards.

## Como usar

1. Leia `docs/roadmap.md` para confirmar a ordem ativa.
2. Use este índice para localizar o bloco certo.
3. Abra apenas o shard do bloco necessário.
4. Consulte `docs/history/` apenas quando precisar da crônica factual detalhada.
