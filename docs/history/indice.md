# Índice do histórico

- **Classe:** Ponte
- **Papel:** navegação
- **Status:** ativo

Este hub organiza o histórico factual da Pinker em descoberta progressiva.

## Como navegar

1. Escolha a categoria histórica correta.
2. Abra o `indice.md` da categoria pelo caminho completo.
3. Use o shard da faixa nominal adequada.
4. Só abra o conteúdo interno necessário.
